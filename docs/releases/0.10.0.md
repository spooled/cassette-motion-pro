# Cassette Motion Pro v0.10.0

## Advanced bike landmarks

This release adds an optional advanced mode to Guided Capture while keeping the
simple 4-point workflow intact.

### Changes

- Keeps Basic Guided Capture as the original 4 landmarks.
- Adds **Advanced landmarks (8 points)**:
  - Bottom bracket center
  - Saddle top
  - Saddle tip
  - Grip / hood contact point
  - Pedal spindle
  - Handlebar center
  - Front axle
  - Rear axle
- Advanced mode calculates additional existing Bike Metrics fields:
  - Crank length
  - Handlebar reach
  - Handlebar drop
- Adds a wheelbase preview in Guided Capture results.
- Keeps v0.9.9 on-image prompts, crosshair cue, larger landmark labels, and
  scrollable controls.
- Updates build, installer, app, and report versions to `0.10.0`.

### Test checklist

1. Open Bike Fit Workspace.
2. Go to Bike Metrics.
3. Choose a Measurement image.
4. Open Guided Capture.
5. Leave Advanced landmarks unchecked and confirm Basic mode still asks for 4
   points.
6. Check Advanced landmarks and confirm it asks for 8 points.
7. Complete all 8 points and confirm Crank length, Handlebar reach, and
   Handlebar drop appear.
8. Save to Before or After.
