# Cassette Motion Pro 0.6.0

This release adds guided bike measurement workflows.

## Changes

- Adds a **Bike Measurement Guide** tab to the Bike Fit Workspace.
- Adds guided instructions for saddle height, saddle setback, saddle-tip-to-grip reach, Handlebar X/Y, handlebar reach, and handlebar drop.
- Adds **Open Before** and **Open After** buttons for each guided measurement.
- Saves the session before opening the selected video.
- Keeps the existing Measurements tab as the place where values are entered and saved.

## Test checklist

1. Open a client.
2. Open the Bike Fit Workspace.
3. Open the **Bike Measurement Guide** tab.
4. Click **Open Before** for Saddle height.
5. Confirm the instruction message appears.
6. Confirm the Before video opens.
7. Repeat with **Open After**.
8. Enter a value in the Measurements tab and confirm it saves.
