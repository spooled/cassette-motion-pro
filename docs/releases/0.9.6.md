# Cassette Motion Pro v0.9.6

## Saddle setback convention

This release updates Guided Capture so saddle setback matches the preferred bike
fit convention.

### Changes

- Changes saddle setback calculation:
  - Behind BB = negative
  - In front of BB = positive
- Updates Guided Capture preview text to show the convention.
- Keeps **Flip Setback Sign** as a manual override.
- Keeps v0.9.5 Guided Capture Accuracy and Level Reference.
- Updates build, installer, app, and report versions to `0.9.6`.

### Test checklist

1. Complete Guided Capture with a saddle tip behind the BB.
2. Confirm saddle setback is negative.
3. Confirm preview says `Behind BB = negative`.
4. Confirm Flip Setback Sign still works.
