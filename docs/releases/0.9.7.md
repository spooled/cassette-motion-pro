# Cassette Motion Pro v0.9.7

## Guided Capture report trace

This release stores Guided Capture metadata and includes it in generated reports.

### Changes

- Adds session fields for Guided Capture trace data.
- Saves trace data separately for Before and After.
- Records:
  - Capture method
  - Level reference status
  - Saddle setback convention
- Adds a **Measurement Capture Trace** table to the report when trace data exists.
- Keeps v0.9.6 saddle setback convention.
- Updates build, installer, app, and report versions to `0.9.7`.

### Test checklist

1. Use Guided Capture and save to Before or After.
2. Generate a report.
3. Confirm the report shows Measurement Capture Trace.
4. Confirm the trace says Guided Capture and Behind BB = negative.
