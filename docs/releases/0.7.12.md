# Cassette Motion Pro v0.7.12

## Report Before measurements toggle

This update adds control over whether **Before** measurements appear in generated reports.

### Changes

- Adds a **Show Before measurements in report** checkbox in the Bike Fit Workspace footer.
- When checked, reports show **Before / After / Change** tables.
- When unchecked, reports show **Final / After** values only.
- Saves the setting per fit session.
- Existing sessions default to showing Before measurements.
- Updates build and installer version to `0.7.12`.

### Test checklist

1. Open a client and fit session.
2. Confirm **Show Before measurements in report** is checked by default.
3. Generate a report and confirm the tables show Before, After, and Change.
4. Uncheck **Show Before measurements in report**.
5. Generate another report and confirm the tables only show Final / After values.
6. Close and reopen the session and confirm the checkbox state is remembered.
