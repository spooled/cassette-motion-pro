# Cassette Motion Pro 0.7.2

This release adds the first real click-to-measure Bike Metrics workflow.

## What changed

- Keeps the v0.7.1 one-button Before + After image combine workflow.
- Replaces the Image Measurement Assistant placeholder buttons with a real
  guided measuring flow.
- Lets the fitter click two calibration points and enter the known real-world
  length in millimeters.
- Lets the fitter click two measurement points on the reference image.
- Calculates the measured value in millimeters.
- Saves the calculated value directly back to the selected Bike Metrics row as
  either Before or After.

## Notes

This is intentionally a simple two-point measurement foundation. More guided
templates, named landmarks, and automatic point transfer can build on this
workflow in later releases.
