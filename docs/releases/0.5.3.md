# Cassette Motion Pro 0.5.3

This release adds side-by-side report image support.

## Changes

- Adds a **Side-by-side image** selector to the Report Images tab.
- Saves the side-by-side image with the fit session.
- Copies the selected side-by-side image into the client Photos folder.
- Shows the side-by-side image full-width in generated reports.
- Keeps the existing Before and After report image slots.

## Test checklist

1. Open a client.
2. Open the Bike Fit Workspace.
3. Open the **Report Images** tab.
4. Choose a side-by-side image.
5. Click **Generate Report**.
6. Open the report and confirm the side-by-side image appears full-width.
7. Reopen the session and confirm the side-by-side image path is still saved.
