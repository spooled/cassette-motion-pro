# Cassette Motion Pro v0.8.4

## Windows bundle artifact

This release makes the GitHub Actions downloads clearer by adding one combined
Windows artifact.

### Changes

- Adds `CassetteMotionPro-Windows-0.8.4`.
- The Windows bundle contains:
  - `CassetteMotionPro-Portable-0.8.4.zip`
  - `CassetteMotionPro-Installer-0.8.4.exe`
- Keeps separate portable and installer artifacts.
- Keeps v0.8.3 Bike Metrics axis defaults.
- Keeps the installer upload staging fix.
- Updates build, installer, app, and report versions to `0.8.4`.

### Test checklist

1. Open the newest successful GitHub Actions run.
2. Download `CassetteMotionPro-Windows-0.8.4`.
3. Confirm the downloaded bundle contains the portable zip and installer exe.
4. Run the installer on Windows.
5. Confirm Cassette Motion Pro opens.
