# Cassette Motion Pro v0.9.3

## Guided Capture polish

This release makes the Guided Capture workflow easier to use and safer to save.

### Changes

- Adds a larger current-point instruction box.
- Adds **Undo Last Point**.
- Adds **Flip Setback Sign**.
- Adds a confirmation preview before saving to Before or After.
- Keeps save buttons disabled until all four landmarks are clicked.
- Cleans up a duplicate image-control registration in the guided form.
- Keeps v0.9.2 installer workflow hardening.
- Updates build, installer, app, and report versions to `0.9.3`.

### Test checklist

1. Open Guided Capture.
2. Calibrate scale.
3. Click one landmark and use Undo Last Point.
4. Complete all four landmarks.
5. Flip saddle setback sign.
6. Save to Before or After and confirm the preview appears.
7. Confirm Bike Metrics fields are updated.
