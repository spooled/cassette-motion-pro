# Cassette Motion Pro 0.7.5

This release adds a horizontal-only measuring mode for Saddle Setback.

## What changed

- Keeps the v0.7.4 side-by-side image workflow.
- Saddle Setback Assist now measures horizontal distance only.
- Other Bike Metrics continue using normal point-to-point distance.
- The **Make Negative / Make Positive** button still works for saddle setback.
- The visual measurement line displays horizontally for saddle setback, matching
  the way the value is calculated.

## Why it matters

Saddle setback is not a diagonal measurement. It should be measured horizontally
from the bottom bracket reference line to the saddle nose, and it may need to be
saved as a negative value depending on fit convention.
