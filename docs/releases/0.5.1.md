# Cassette Motion Pro 0.5.1

This release polishes the first report workflow.

## Changes

- Adds **Open Reports** to the Bike Fit Workspace.
- Adds a **Print / Save PDF** button inside the generated HTML report.
- Adds Before and After image placeholders to the report.
- Adds a **Change** column to measurement and body-angle tables when before/after values can be compared.
- Improves report print styling for easier browser PDF export.

## Test checklist

1. Open a client.
2. Open the Bike Fit Workspace.
3. Enter before/after values such as `742 mm` and `748 mm`.
4. Click **Generate Report**.
5. Click **Open Reports**.
6. Open the report and confirm the Change column appears.
7. Click **Print / Save PDF** in the browser and confirm the print dialog opens.
