# Cassette Motion Pro 0.7.1

This release adds one-button Before + After image combine for reports and Bike
Metrics.

## What changed

- Added a **Combine Before + After** button to the Report Images tab.
- Added a **Combine B+A** button beside the Bike Metrics measurement image tools.
- Creates a side-by-side JPG from the selected before and after images.
- Saves the combined image inside the current client's Photos folder.
- Automatically fills the Side-by-side report image field.
- Automatically uses the combined image as the Bike Metrics Measurement
  Reference Image.

## Why it matters

Fitters can now choose the before and after still images once, click one button,
and immediately use the combined image in both the report and the guided Bike
Metrics assistant.
