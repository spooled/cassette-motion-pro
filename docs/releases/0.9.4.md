# Cassette Motion Pro v0.9.4

## Installer NSIS hardening

This release changes how GitHub Actions installs and calls NSIS.

### Changes

- Replaces the third-party NSIS install action with Chocolatey.
- Locates the real `makensis.exe` path on the Windows runner.
- Calls NSIS using that exact executable path.
- Records the NSIS path in `INSTALLER-BUILD-STATUS.txt`.
- Keeps the Windows bundle status-file workflow.
- Keeps v0.9.3 Guided Capture polish.
- Updates build, installer, app, and report versions to `0.9.4`.

### Test checklist

1. Open the newest GitHub Actions run.
2. Download `CassetteMotionPro-Windows-0.9.4`.
3. Open `INSTALLER-BUILD-STATUS.txt`.
4. Confirm status is `SUCCESS`.
5. Confirm `CassetteMotionPro-Installer-0.9.4.exe` is inside the Windows bundle.
