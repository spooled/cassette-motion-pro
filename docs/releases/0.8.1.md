# Cassette Motion Pro v0.8.1

## Measurement axis lock

This update improves the Image Measurement Assistant for bike metrics.

### Changes

- Adds a **Measurement** mode selector:
  - Free
  - Horizontal
  - Vertical
- Holding **Shift** on the second measurement click auto-locks the line:
  - mostly left/right movement becomes horizontal
  - mostly up/down movement becomes vertical
- Horizontal measurements use X distance only.
- Vertical measurements use Y distance only.
- Free measurements keep normal point-to-point distance.
- Existing negative/manual measurement support is preserved.
- Saddle setback still defaults to horizontal mode.
- Updates build, installer, app, and report versions to `0.8.1`.

### Test checklist

1. Open Bike Fit Workspace.
2. Go to Bike Metrics.
3. Open an image measurement assistant.
4. Calibrate scale.
5. Set mode to **Free** and measure a diagonal distance.
6. Set mode to **Horizontal** and confirm the line snaps horizontally.
7. Set mode to **Vertical** and confirm the line snaps vertically.
8. Set mode to **Free**, hold **Shift** on the second click, and confirm it auto-locks horizontal or vertical based on point direction.
9. Confirm negative values and manual values still save.
