# Cassette Motion Pro v0.10.2

## Wheelbase Bike Metric

This release gives wheelbase a permanent place in Bike Metrics and generated
reports.

### Changes

- Adds **Wheelbase** as a Before/After Bike Metrics row.
- Saves and reloads manual Wheelbase values in the fit session.
- Advanced Guided Capture saves calculated Wheelbase into Bike Metrics.
- Adds Wheelbase to the generated report.
- Sets Wheelbase image-assist to horizontal measurement by default.
- Keeps v0.10.1 editable Guided Capture landmarks.
- Updates build, installer, app, and report versions to `0.10.2`.

### Test checklist

1. Open Bike Fit Workspace.
2. Go to Bike Metrics.
3. Confirm there is a Wheelbase row.
4. Type a manual Wheelbase value, save, close, and reopen.
5. Confirm the Wheelbase value reloads.
6. Use Advanced Guided Capture.
7. Confirm Wheelbase appears in calculated values.
8. Save to Before or After.
9. Generate a report and confirm Wheelbase appears.
