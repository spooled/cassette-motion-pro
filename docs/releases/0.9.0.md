# Cassette Motion Pro v0.9.0

## Guided landmark capture

This release adds the first multi-measurement guided Bike Metrics workflow.

### Changes

- Adds **Guided Capture** to the Bike Metrics measurement image controls.
- Opens a guided landmark form using the selected measurement reference image.
- Supports zoom, pan, image centering, and scale calibration.
- Guides the fitter through four clicks:
  - Bottom bracket center
  - Saddle top
  - Saddle tip
  - Grip / hood contact point
- Calculates:
  - Saddle height
  - Saddle setback
  - Saddle tip to grip reach
  - Handlebar X
  - Handlebar Y
- Saves the calculated set to either Before or After.
- Keeps individual field-level Assist buttons.
- Keeps the Windows bundle artifact.
- Updates build, installer, app, and report versions to `0.9.0`.

### Test checklist

1. Choose a measurement reference image.
2. Open Guided Capture.
3. Calibrate the scale.
4. Click the four requested landmarks.
5. Save to Before.
6. Reopen and save to After.
7. Confirm Bike Metrics fields update and save with the session.
