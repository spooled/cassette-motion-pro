# Cassette Motion Pro v0.9.9

## Guided Capture clarity

This release makes Guided Capture easier to use while clicking bike measurement
landmarks.

### Changes

- Keeps the v0.9.8 scrollable Guided Capture side panel.
- Adds a clearer “next point” hint under the current point.
- Adds an on-image prompt while calibration, level reference, or landmark
  capture is active.
- Adds a highlighted crosshair around the mouse while capture mode is active.
- Makes clicked landmark dots, labels, and guide lines easier to see.
- Keeps Guided Capture report trace behavior.
- Updates build, installer, app, and report versions to `0.9.9`.

### Test checklist

1. Open Bike Fit Workspace.
2. Go to Bike Metrics.
3. Choose a Measurement image.
4. Open Guided Capture.
5. Calibrate Scale and optionally set Level Reference.
6. Start Guided Capture.
7. Confirm the right panel says exactly which point to click next.
8. Confirm the on-image prompt and mouse crosshair are visible.
9. Click all four landmarks and confirm the labels are easier to read.
10. Save to Before or After.
