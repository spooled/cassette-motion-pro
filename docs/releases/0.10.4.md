# Cassette Motion Pro v0.10.4

## Installer resource hardening

This release fixes installer builds when the old NSIS custom graphics/license
resource folders are not present in the GitHub upload.

### Changes

- Keeps v0.10.3 Camera Accuracy Guide.
- Removes dependency on missing `src/Installer/graphics` files.
- Removes dependency on missing `src/Installer/OtherFiles/GPLv2.txt`.
- Adds installer source file listing to the GitHub Actions log before NSIS runs.
- Keeps installer status file in the Windows bundle.
- Updates build, installer, app, and report versions to `0.10.4`.

### Test checklist

1. Upload this package to GitHub.
2. Wait for Actions to finish.
3. Download `CassetteMotionPro-Windows-0.10.4`.
4. Confirm `INSTALLER-BUILD-STATUS.txt` says `SUCCESS`.
5. Confirm `CassetteMotionPro-Installer-0.10.4.exe` is present.
6. Run the installer on Windows.
