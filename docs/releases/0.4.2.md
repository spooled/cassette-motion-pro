# Cassette Motion Pro 0.4.2

This release makes the bike-fit workspace safer to use during real testing.

## Changes

- Saves the current fit session automatically when the workspace is closed.
- Renames the footer action to **Save & Close** so the behavior is clearer.
- Keeps manual **Save Session** for deliberate saves while staying in the workspace.
- Updates the footer message to explain that sessions are saved in the selected client’s Measurements folder.
- Keeps the v0.4.1 body-angle overlay activation fix intact.

## Test checklist

1. Open a client.
2. Open the bike-fit workspace.
3. Enter a few measurements or notes.
4. Click **Save & Close**.
5. Reopen the same client and workspace.
6. Confirm the saved session and values are still there.
7. Confirm **Measure Before Video** and **Measure After Video** still open the body-angle guide.
