# Cassette Motion Pro v0.9.5

## Guided Capture accuracy

This release improves Guided Capture calculations by adding an optional level
reference line.

### Changes

- Adds **Level Reference** inside Guided Capture.
- Uses a two-point floor/axle/reference line to correct tilted images.
- Applies level correction to horizontal/vertical calculations:
  - Saddle setback
  - Saddle tip to grip reach
  - Handlebar X
  - Handlebar Y
- Keeps saddle height as true angled `Distance` from BB center to saddle top.
- Adds **Recalculate Values**.
- Adds preview details for level-reference status and saddle-setback convention.
- Keeps v0.9.4 installer NSIS hardening.
- Updates build, installer, app, and report versions to `0.9.5`.

### Test checklist

1. Calibrate scale.
2. Set a level reference line.
3. Complete the four Guided Capture landmarks.
4. Confirm the preview says `Level reference: Applied`.
5. Click Recalculate Values.
6. Save to Before or After.
7. Confirm Bike Metrics fields update.
