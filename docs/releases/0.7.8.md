# Cassette Motion Pro v0.7.8

## Zoom centering for image measurements

This update improves the Image Measurement Assistant zoom workflow.

### Changes

- Adds a **Center Image** button beside the zoom controls.
- Keeps the image centered when it fits inside the measurement window.
- Prevents panning the zoomed image too far off-screen.
- Preserves the v0.7.7 signed manual measurement update, including negative values such as `-9.0 mm`.

### Test checklist

1. Open the Bike Fit Workspace.
2. Open Bike Metrics.
3. Choose an image for a metric such as Saddle Setback.
4. Zoom in using the mouse wheel or `+`.
5. Pan with right-click drag or middle-click drag.
6. Click **Center Image**.
7. Confirm the photo recenters while staying at the same zoom level.
8. Confirm measurement clicks still land accurately while zoomed.
