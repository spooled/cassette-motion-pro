# Cassette Motion Pro 0.7.3

This release adds signed measurement support for Bike Metrics values such as
saddle setback.

## What changed

- Keeps the v0.7.2 click-to-measure Bike Metrics assistant.
- Adds a **Make Negative / Make Positive** button after measuring.
- Allows measured values to be saved below zero.
- Saves the signed result directly to the selected Bike Metrics row as Before
  or After.

## Why it matters

Some fit measurements are directional. Saddle setback, for example, may need to
show a negative value depending on how the fitter defines the bottom bracket
reference line and saddle position.
