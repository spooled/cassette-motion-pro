# Cassette Motion Pro v0.8.2

## Installer upload staging fix

This release keeps the v0.8.1 measurement tools and fixes the GitHub Actions
packaging path for the Windows installer artifact.

### Changes

- Updates build, installer, app, and report versions to `0.8.2`.
- Builds the NSIS installer as before.
- Searches for the generated `CassetteMotionPro-*.exe` after the NSIS build.
- Copies the installer into `dist/installer/`.
- Uploads the installer artifact from `dist/installer/*.exe`.
- Fails the workflow clearly if no installer executable is found.

### Test checklist

1. Open the newest successful GitHub Actions run.
2. Confirm both artifacts are visible:
   - `CassetteMotionPro-Portable-0.8.2`
   - `CassetteMotionPro-Installer-0.8.2`
3. Download the portable build and confirm the app still opens.
4. Download the installer build and confirm it installs on Windows.
