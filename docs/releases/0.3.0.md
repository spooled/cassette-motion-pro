# Cassette Motion Pro 0.3.0

Version 0.3 introduces the Bike Fit Workspace, a client-centered home for each
fit session.

## Bike Fit Workspace

- Create and reopen persistent fit sessions for every client
- Track rider goals, session status, recommendations, and follow-up notes
- Organize side, front, rear, before, and after videos
- Import selected videos automatically into the client's Videos folder
- Launch individual videos in the existing analysis engine
- Launch side/front or before/after pairs in the synchronized dual player
- Record before and after saddle height, saddle setback, handlebar reach,
  handlebar drop, crank length, and cleat position

Session records are stored under the client's `Measurements/Sessions` folder.
Imported media is stored under `Videos/Fit Sessions` and remains available even
if the original source video is moved.
