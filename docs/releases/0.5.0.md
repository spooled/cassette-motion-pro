# Cassette Motion Pro 0.5.0

This release adds the first report generator.

## Changes

- Adds **Generate Report** to the Bike Fit Workspace.
- Saves the current session before creating the report.
- Creates a branded HTML report in the selected client’s Reports folder.
- Includes client, bike, session date, status, rider goals, before/after bike measurements, body angles, and notes.
- Keeps v0.4.2 safer workspace saving.

## Test checklist

1. Open a client.
2. Open the Bike Fit Workspace.
3. Enter a few measurements and notes.
4. Click **Generate Report**.
5. Confirm the app says the report was saved.
6. Open the client’s Reports folder.
7. Open the `.html` report and confirm the values are listed.
