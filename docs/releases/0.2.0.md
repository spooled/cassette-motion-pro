# Cassette Motion Pro 0.2.0

Version 0.2 introduces the first bike-fit-specific workflow on top of the
Kinovea engine.

## Client Manager

- Create persistent client records with contact information and notes
- Record bicycle make, model, year, and type
- Search clients by name, bicycle, email, or phone
- Open recent clients directly from the new Clients menu
- Open a selected client's Videos folder in the existing media browser
- Open the complete client folder in Windows Explorer

Each new client receives this structure automatically:

```text
Client_Name_ID/
├── Videos/
├── Photos/
├── Reports/
├── Measurements/
├── Notes/
└── client.xml
```

Client information is stored under the Cassette Motion Pro application-data
folder and remains separate from Kinovea's video engine.
