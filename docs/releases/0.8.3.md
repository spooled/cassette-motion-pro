# Cassette Motion Pro v0.8.3

## Bike metric axis defaults

This release makes the Image Measurement Assistant choose better default
distance modes for Bike Metrics.

### Changes

- Adds a default measurement axis setting to the Image Measurement Assistant.
- Opens **Saddle setback** in Horizontal mode.
- Opens **Handlebar X** in Horizontal mode.
- Opens **Handlebar Y** in Vertical mode.
- Opens **Saddle tip to grip reach** in Horizontal mode.
- Keeps all other metric assists in Free mode unless changed manually.
- Keeps Shift-click auto-lock behavior.
- Keeps the v0.8.2 installer artifact staging fix.
- Updates build, installer, app, and report versions to `0.8.3`.

### Test checklist

1. Open Bike Fit Workspace.
2. Open Bike Metrics.
3. Choose a measurement reference image.
4. Open Assist for Saddle setback and confirm it starts in Horizontal mode.
5. Open Assist for Handlebar X and confirm it starts in Horizontal mode.
6. Open Assist for Handlebar Y and confirm it starts in Vertical mode.
7. Open Assist for Saddle tip to grip reach and confirm it starts in Horizontal mode.
8. Confirm manual mode switching still works.
