# Cassette Motion Pro 0.7.0

This release adds the foundation for the image-based measurement assistant.

## Changes

- Adds an **Image Measurement Assistant** screen.
- Opens the selected Measurement Reference Image from Bike Metrics **Assist**.
- Displays the selected bike metric and how-to-measure instructions.
- Shows planned steps for calibration, landmark clicking, and saving the metric.
- Keeps calculation and automatic save-back as future work.

## Test checklist

1. Open a client.
2. Open the Bike Fit Workspace.
3. Open Bike Metrics.
4. Choose a Measurement image.
5. Click **Assist** on Saddle height or another metric.
6. Confirm the image assistant opens and displays the image.
7. Confirm the metric instructions appear.
8. Click the planned step buttons and confirm the “Coming soon” message appears.
