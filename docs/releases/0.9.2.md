# Cassette Motion Pro v0.9.2

## Installer workflow hardening

This release improves GitHub Actions packaging so installer problems are easier
to diagnose and less likely to hide the Windows download bundle.

### Changes

- Uploads `CassetteMotionPro-Windows-0.9.2` even if installer creation fails.
- Adds `INSTALLER-BUILD-STATUS.txt` to the Windows bundle.
- Includes installer success/failure details in that status file.
- Keeps the portable zip in the Windows bundle.
- Keeps the installer exe in the Windows bundle when NSIS succeeds.
- Passes the installer version into NSIS from the workflow version.
- Keeps v0.9.1 Distance / Distance horizontal / Distance vertical labels.
- Updates build, installer, app, and report versions to `0.9.2`.

### Test checklist

1. Open the newest GitHub Actions run.
2. Download `CassetteMotionPro-Windows-0.9.2`.
3. Confirm `INSTALLER-BUILD-STATUS.txt` is present.
4. If the status says `SUCCESS`, confirm the installer exe is present.
5. If the status says `FAILED`, inspect the status file and the Build installer
   log.
