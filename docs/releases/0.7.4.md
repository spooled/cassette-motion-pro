# Cassette Motion Pro 0.7.4

This release makes side-by-side-only image workflows easier.

## What changed

- Keeps the v0.7.3 negative Bike Metrics measurement support.
- Lets the fitter use a single side-by-side image without separate Before and
  After images.
- Automatically sets the chosen Side-by-side report image as the Bike Metrics
  Measurement image.
- Keeps the Combine Before + After button available for cases where the fitter
  wants the app to create the side-by-side image.

## Why it matters

If a fitter already exported or captured a clean side-by-side image, they can
use that one image directly for both reports and Bike Metrics measuring.
