# Cassette Motion Pro 0.5.2

This release adds before/after report images.

## Changes

- Adds a **Report Images** tab to the Bike Fit Workspace.
- Adds **Before image** and **After image** selectors.
- Copies selected images into the client Photos folder.
- Saves report image paths with the fit session.
- Shows the selected before/after images in generated HTML reports.

## Test checklist

1. Open a client.
2. Open the Bike Fit Workspace.
3. Open the **Report Images** tab.
4. Browse for a before image and an after image.
5. Click **Generate Report**.
6. Open the report and confirm both images appear.
7. Reopen the same session and confirm the image paths are still saved.
