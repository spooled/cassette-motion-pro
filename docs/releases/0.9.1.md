# Cassette Motion Pro v0.9.1

## Distance tool labels

This release clarifies measurement tool wording in the Image Measurement
Assistant.

### Changes

- Renames user-facing measurement modes:
  - `Distance`
  - `Distance (horizontal)`
  - `Distance (vertical)`
- Clarifies Saddle height as an angled `Distance` measurement from bottom
  bracket center to saddle top along the saddle-height line.
- Keeps field defaults:
  - Saddle height: `Distance`
  - Saddle setback: `Distance (horizontal)`
  - Saddle tip to grip reach: `Distance (horizontal)`
  - Handlebar X: `Distance (horizontal)`
  - Handlebar Y: `Distance (vertical)`
- Keeps Guided Capture from v0.9.0.
- Updates build, installer, app, and report versions to `0.9.1`.

### Test checklist

1. Open Saddle height Assist and confirm it starts on `Distance`.
2. Draw an angled BB-to-saddle-top line.
3. Open Saddle setback Assist and confirm it starts on `Distance (horizontal)`.
4. Open Handlebar Y Assist and confirm it starts on `Distance (vertical)`.
5. Confirm Guided Capture still opens and saves values.
