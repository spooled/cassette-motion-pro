# Cassette Motion Pro v0.8.0

## Report polish

This update makes generated fit reports feel more professional and client-ready.

### Changes

- Redesigns the report header with a cleaner professional bike-fit layout.
- Adds session summary cards for session, date, status, and report view.
- Improves report image presentation with darker image panels and contained image scaling.
- Replaces the generic report flow with clearer bike-fit sections:
  - Visual Fit Review
  - Measurement Reference Image
  - Bike Measurements
  - Contact points
  - Handlebar position
  - Foot interface
  - Body Angles
  - Recommendations and Notes
- Keeps report toggles for Before measurements and image display.
- Updates build, installer, app, and report versions to `0.8.0`.

### Test checklist

1. Build with GitHub Actions.
2. Confirm artifacts are named:
   - `CassetteMotionPro-Portable-0.8.0`
   - `CassetteMotionPro-Installer-0.8.0`
3. Generate a report with Before measurements turned on.
4. Generate a report with Before measurements turned off.
5. Toggle report image display options and confirm the generated report follows your choices.
6. Use Print / Save PDF from the generated report and confirm the layout looks clean.
