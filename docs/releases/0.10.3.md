# Cassette Motion Pro v0.10.3

## Camera Accuracy Guide

This release adds a camera setup checklist to Guided Capture and records the
setup status in generated reports.

### Changes

- Adds a **Camera Setup** button at the start of Guided Capture.
- Checklist covers:
  - Camera straight side-on to bike
  - Camera level
  - Bike upright
  - 2x/telephoto or farther-back setup when possible
  - Avoiding ultra-wide lens distortion
  - Calibration length in the same plane as the bike
  - Level Reference when the image is tilted
- Saves Camera setup as **Confirmed** or **Not confirmed**.
- Adds Camera setup to the report Measurement Capture Trace.
- Keeps v0.10.2 wheelbase behavior.
- Updates build, installer, app, and report versions to `0.10.3`.

### Test checklist

1. Open Guided Capture.
2. Click **Camera Setup**.
3. Confirm the checklist appears.
4. Click **Yes**.
5. Complete Guided Capture and save to Before or After.
6. Generate a report.
7. Confirm Measurement Capture Trace shows **Camera setup: Confirmed**.
