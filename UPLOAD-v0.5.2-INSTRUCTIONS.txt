Cassette Motion Pro v0.5.2
==========================

This update adds before/after images to reports.

What changed:
- Adds a Report Images tab to the Bike Fit Workspace.
- Adds Before image and After image selectors.
- Copies chosen images into the client Photos folder.
- Saves those image choices with the fit session.
- Shows the images in generated HTML reports.

Upload steps:

1. Extract Cassette-Motion-Pro-0.5.2-report-images.zip.
2. Open the extracted folder.
3. Upload everything INSIDE the extracted folder to the GitHub repository root.
4. Commit with: Add v0.5.2 report images

Then edit .github/workflows/build.yml and change VERSION to 0.5.2. Commit the
change and wait for the Actions run to receive a green check.

Test checklist:

1. Open a client.
2. Open the Bike Fit Workspace.
3. Open the Report Images tab.
4. Browse for a before image and an after image.
5. Click Generate Report.
6. Open the report and confirm both images appear.
7. Reopen the session and confirm the image paths are still there.
